## ✨ Sobre o Portfólio

Este não é apenas um portfólio — é um diário de jornada.  
Sou a Fernanda Romão, designer gráfica de formação, desenvolvedora por escolha, aprendiz de mandarim por amor.

**Mokshyaa** é o nome que dei a essa jornada — uma palavra nascida da junção do sânscrito *moksha* e o mandarim 花 *(huā)*. A flor que desabrocha na escuridão.
