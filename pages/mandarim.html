变量 biànliàng
Variável Variable

Obs: também pode significar "inconstante/mutável" no sentido figurado.

  • 我声明了一个变量来存储用户名。

    Wǒ shēngmíngle yīgè biànliàng lái cúnchǔ yònghùmíng.

    Declarei uma variável para armazenar o nome do usuário.

  • 这个变量的值是空的。

    Zhège biànliàng de zhí shì kōng de.

    O valor dessa variável está vazio.

函数 hánshù
Função Function

Obs: termo formal; informalmente 功能 (gōngnéng) é usado para "funcionalidade".

  • 这个函数返回用户的年龄。

    Zhège hánshù fǎnhuí yònghù de niánlíng.

    Essa função retorna a idade do usuário.

  • 你调用了错误的函数。

    Nǐ diàoyòngle cuòwù de hánshù.

    Você chamou a função errada.

循环 xúnhuán
Loop Loop

Obs: também significa "ciclo" em contextos naturais e médicos.

  • 这个循环会一直运行到条件为假。

    Zhège xúnhuán huì yīzhí yùnxíng dào tiáojiàn wèi jiǎ.

    Esse loop vai rodar até a condição ser falsa.

  • 小心无限循环！

    Xiǎoxīn wúxiàn xúnhuán!

    Cuidado com loop infinito!

算法 suànfǎ
Algoritmo Algorithm

Obs: usado tanto no contexto técnico quanto no cotidiano para descrever "lógica de raciocínio".

  • 我们需要优化这个算法。

    Wǒmen xūyào yōuhuà zhège suànfǎ.

    Precisamos otimizar esse algoritmo.

  • 这个算法的时间复杂度是O(n)。

    Zhège suànfǎ de shíjiān fùzádù shì O(n).

    A complexidade de tempo desse algoritmo é O(n).

漏洞 lòudòng
Bug Bug / Vulnerability

Obs: termo formal; também significa "brecha" em segurança. "bug" em inglês também é amplamente usado.

  • 我找到了一个很难发现的漏洞。

    Wǒ zhǎodàole yīgè hěn nán fāxiàn de lòudòng.

    Encontrei um bug muito difícil de achar.

  • 这个bug让整个系统崩溃了。

    Zhège bug ràng zhěnggè xìtǒng bēngkuìle.

    Esse bug derrubou o sistema inteiro.

调试 tiáoshì
Depurar Debug

Obs: também usado para "calibrar/ajustar" equipamentos fora do contexto de programação.

  • 我花了三个小时调试这段代码。

    Wǒ huāle sān gè xiǎoshí tiáoshì zhè duàn dàimǎ.

    Passei três horas depurando esse código.

  • 调试是开发过程中最重要的技能之一。

    Tiáoshì shì kāifā guòchéng zhōng zuì zhòngyào de jìnéng zhīyī.

    Depurar é uma das habilidades mais importantes no desenvolvimento.

编译 biānyì
Compilar Compile

Obs: não confundir com 翻译 (fānyì) que significa "traduzir".

  • 代码编译失败了。

    Dàimǎ biānyì shībàile.

    A compilação do código falhou.

  • 编译需要几分钟。

    Biānyì xūyào jǐ fēnzhōng.

    A compilação vai levar alguns minutos.

条件 tiáojiàn
Condição Condition

Obs: também significa "requisito" ou "circunstância" no cotidiano — muito versátil.

  • 如果条件为真，执行这段代码。

    Rúguǒ tiáojiàn wèi zhēn, zhíxíng zhè duàn dàimǎ.

    Se a condição for verdadeira, execute esse código.

  • 这个条件永远不会满足。

    Zhège tiáojiàn yǒngyuǎn bù huì mǎnzú.

    Essa condição nunca vai ser satisfeita.

网络用语 · Gírias de Internet

666 liù liù liù
Incrível / Que demais Amazing / So good

Obs: o número 6 (liù) soa como "fluir/fluente" — triplicado vira elogio máximo.

  • 你的代码写得666！

    Nǐ de dàimǎ xiě de 666!

    Seu código está incrível!

  • 这个操作真的666。

    Zhège cāozuò zhēn de 666.

    Essa jogada foi demais.

886 bā bā liù
Tchau / Até mais Bye bye

Obs: 8=bā (soa como "ba" de bye), 6=liù (soa como "liu" de leave). Despedida informal de chat.

  • 我要去睡了，886！

    Wǒ yào qù shuìle, 886!

    Vou dormir, tchau!

  • 下线了，886～

    Xià xiàn le, 886～

    Saindo do ar, até mais!

菜鸟 càiniǎo
Noob / Iniciante Newbie

Obs: literalmente "pássaro vegetal/frango" — pode ser carinhoso ou zoação dependendo do contexto.

  • 我刚开始学编程，还是个菜鸟。

    Wǒ gāng kāishǐ xué biānchéng, háishì gè càiniǎo.

    Acabei de começar a aprender programação, ainda sou um noob.

  • 别嘲笑菜鸟，我们都是从零开始的。

    Bié cháoxiào càiniǎo, wǒmen dōu shì cóng líng kāishǐ de.

    Não zoe os noobs, todos começamos do zero.

orz orz
Me rendo / Derrotado Defeated / Bowing down

Obs: formato visual de pessoa de joelhos — derrota, rendição ou admiração extrema.

  • 这道题太难了，orz。

    Zhè dào tí tài nánle, orz.

    Essa questão é difícil demais, me rendo.

  • 你居然一次就通过了？orz！

    Nǐ jūrán yīcì jiù tōngguòle? orz!

    Passou de primeira?! Me curvo.

卡顿 kǎdùn
Lag / Travar Lag / Stutter

Obs: coloquial para travar/engasgar; 延迟 (yánchí) é o termo técnico para latência.

  • 游戏又卡顿了，太烦了！

    Yóuxì yòu kǎdùnle, tài fánle!

    O jogo travou de novo, que raiva!

  • 网络延迟太高，根本没法玩。

    Wǎngluò yánchí tài gāo, gēnběn méifǎ wán.

    O lag está alto demais, impossível jogar.

挂机 guàjī
AFK Away from keyboard

Obs: literalmente "deixar a máquina pendurada/rodando" — muito usado em games.

  • 我要挂机去吃饭。

    Wǒ yào guàjī qù chīfàn.

    Vou dar AFK para comer.

  • 队友一直挂机，害我们输了。

    Duìyǒu yīzhí guàjī, hài wǒmen shūle.

    O colega ficou AFK o tempo todo, fez a gente perder.

爆火 bàohuǒ
Viralizar Go viral

Obs: literalmente "explodir em fogo". Também usam 出圈 (chūquān = sair do nicho) para algo que estoura além do público original.

  • 这个视频突然爆火了。

    Zhège shìpín túrán bàohuǒle.

    Esse vídeo viralizou do nada.

  • 她的项目在GitHub上爆火。

    Tā de xiàngmù zài GitHub shàng bàohuǒ.

    O projeto dela viralizou no GitHub.

喷子 pēnzi
Troll / Hater Troll

Obs: literalmente "pessoa que cospe/esguicha" — alguém que só vai para atacar e criticar sem fundamento.

  • 评论区全是喷子。

    Pínglùn qū quán shì pēnzi.

    A seção de comentários está cheia de trolls.

  • 不要理喷子，浪费时间。

    Bùyào lǐ pēnzi, làngfèi shíjiān.

    Não dê atenção a troll, é perda de tempo.
